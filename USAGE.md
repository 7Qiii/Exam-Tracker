# 使用指南

## 快速开始

### 在华为 Mate 70 Pro 上使用

1. **启动本地服务器**（在电脑上）
   ```bash
   npx http-server . -p 5173
   ```

2. **手机访问**
   - 查看电脑的局域网 IP（如 192.168.1.100）
   - 在华为浏览器中访问 `http://192.168.1.100:5173`

3. **添加到桌面**
   - 点击浏览器菜单 → "添加到桌面"
   - 之后可以像原生应用一样打开

## 核心功能说明

### 📝 记录模考

1. 在"新增模考"区域填写信息
2. 选择科目（会自动填入该科目的满分）
3. 填写得分、日期、用时等
4. 选择主要失分原因
5. 添加复盘备注（重要！）
6. 点击"保存记录"

**提示**：
- 用时单位是分钟
- 复盘备注帮助你记住这套卷的核心收获
- 失分原因统计会在饼图中展示

### 📊 查看数据分析

#### 单科趋势图
- 显示最近 8 次成绩
- **虚线**：3 次移动平均线（平滑趋势）
- **橙色虚线**：目标分数线
- 用于判断是否稳定达标

#### 雷达图
- 同时对比所有科目的得分率
- 快速识别强项和弱项
- 面积越大表示整体水平越高

#### 失分原因饼图
- 统计所有记录的失分原因分布
- 找出最常见的问题类型
- 针对性补强

#### 练习频率热力图
- 近 30 天的练习频率
- 颜色越深表示练习越多
- 保持规律练习很重要

### 💡 学习洞察

系统会自动分析你的数据，提供：

- **学习强度评估**：评估你的练习频率
- **进步趋势**：识别哪些科目在进步
- **退步预警**：提醒近期得分下降的科目
- **目标预测**：预测按当前趋势多久能达标
- **问题识别**：找出重复出现的失分原因

### ⚙️ 科目设置

1. 点击"各科状态"右侧的"科目"按钮
2. 修改科目名称、满分、目标分
3. 点击"保存科目设置"

**注意**：修改满分不会影响已有记录

### 💾 数据管理

#### 导出数据
- 点击右上角"↓"按钮
- 自动下载 JSON 文件
- 建议定期备份

#### 导入数据
- 点击右上角"↑"按钮
- 选择之前导出的 JSON 文件
- 选择导入方式：
  - **合并数据**：保留现有记录，添加新记录
  - **覆盖数据**：删除所有现有数据，只保留导入的

**场景示例**：
- 换手机/浏览器：导出 → 在新设备导入（覆盖）
- 合并多设备数据：分别导出 → 在一台设备上导入（合并）

## 高级技巧

### 充分利用复盘备注
好的复盘备注示例：
- ✅ "选择题第 5、8、12 题考察指针和结构体，需要回顾"
- ✅ "大题第 3 题动态规划思路卡壳，看解析后懂了"
- ❌ "做得不好"（信息量太少）

### 解读学习洞察

**"近期得分下降"**
- 可能是遇到了更难的题
- 可能是某个知识点遗忘了
- 建议查看该科目的失分原因分布

**"预计还需 X 次达到目标"**
- 基于线性回归预测
- 仅供参考，不是绝对的
- 如果预测次数很多，说明进步速度较慢

**"主要问题是..."**
- 统计近 5 次的失分原因
- 如果某个原因占比超过 60%，需要重点关注

### 保持学习节奏
- **高强度**（每天 1+ 次）：注意劳逸结合，避免疲劳
- **中等强度**（每天 0.5-1 次）：理想状态，保持即可
- **低强度**（每天 < 0.5 次）：建议增加练习频率

## 数据存储说明

- 所有数据存储在浏览器的 IndexedDB 中
- 清除浏览器数据会删除所有记录
- **重要**：定期导出备份，防止数据丢失
- 数据完全离线，不会上传到任何服务器

## 常见问题

### Q: 数据会同步到云端吗？
A: 不会。所有数据只存储在你的设备上，完全离线。

### Q: 换手机后数据怎么办？
A: 在旧手机上导出数据，在新手机上导入即可。

### Q: 可以修改已有记录吗？
A: 目前只能删除记录，不能修改。建议删除后重新添加。

### Q: 图表显示不全怎么办？
A: 尝试横屏查看，或者在浏览器设置中调整缩放比例。

### Q: 离线能用吗？
A: 可以。首次访问后，Service Worker 会缓存所有资源，之后完全离线可用。

## 建议的使用流程

1. **做完一套卷** → 立即记录成绩和复盘
2. **每周日晚** → 查看本周数据，导出备份
3. **每月初** → 查看雷达图和洞察，调整学习计划
4. **考前一周** → 回顾所有复盘备注，查漏补缺
