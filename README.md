# 11408 备考分数本

个人自用的 11408 考研模考记录 PWA。默认科目为数学一、英语一、政治、408，数据保存在浏览器本地（IndexedDB）。

## 使用

直接打开 `index.html` 可以使用。若要在手机访问，建议在电脑上启动一个本地静态服务，再用手机访问电脑的局域网地址。

```powershell
npx http-server . -p 5173
```

在手机浏览器（推荐使用华为浏览器或 Chrome）中访问后，可以"添加到主屏幕"作为独立应用使用。

## 当前功能

### 基础记录
- 记录模考科目、试卷名称、日期、得分、满分、用时、失分原因和复盘备注
- 首页展示总记录、近 7 天练习次数、平均目标差距
- 各科平均分、最近记录和目标进度
- 科目满分和目标分设置

### 数据可视化（新增）
- **单科趋势图**：显示最近 8 次成绩变化，带移动平均线和目标线
- **雷达图**：各科目得分率对比，一目了然看出强弱项
- **失分原因饼图**：分析失分原因分布，找出主要问题
- **练习频率热力图**：近 30 天练习频率可视化

### 智能分析（新增）
- **学习洞察**：基于数据自动生成学习建议
- **进步趋势分析**：识别各科进步或退步趋势
- **目标达成预测**：预测按当前趋势达到目标所需次数
- **学习强度评估**：分析练习频率，给出节奏建议

### 数据管理（增强）
- **IndexedDB 存储**：替代 localStorage，支持更大数据量
- **数据导入**：支持导入之前导出的 JSON 文件
- **合并模式**：导入时可选择合并或覆盖现有数据
- **数据导出**：JSON 格式导出，方便备份

### 移动端优化（新增）
- 针对华为 Mate 70 Pro 的 HarmonyOS 6 优化
- 触摸友好的交互设计
- 安全区域适配（刘海屏、药丸屏）
- PWA 离线缓存，随时随地使用

## 技术栈

- 原生 JavaScript（ES6 模块）
- IndexedDB 数据持久化
- Canvas 2D 数据可视化
- Service Worker 离线支持
- Progressive Web App

## 功能亮点

✨ **完全离线**：所有数据存储在本地，无需网络即可使用  
📊 **多维分析**：4 种图表类型，多角度分析学习数据  
🎯 **智能建议**：AI 分析学习趋势，提供个性化建议  
📱 **移动优先**：针对手机使用场景优化，支持添加到桌面  
💾 **数据安全**：支持导入导出，数据完全掌控在自己手中
